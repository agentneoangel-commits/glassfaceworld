## Search Tips

### Apple Music
1. Go to music.apple.com
2. Search: "[Artist] [Song]"
3. Right-click album artwork → "Copy Image Address"
4. URL format: `https://is1-ssl.mzstatic.com/image/thumb/.../..._cover.jpg`

### Spotify
1. Open Spotify web player
2. Search song
3. Right-click artwork → "Copy Image Address"  
4. URL format: `https://i.scdn.co/image/...`

### YouTube (for videos)
- Thumbnail: `https://img.youtube.com/vi/{VIDEO_ID}/hqdefault.jpg`
- Max res: `https://img.youtube.com/vi/{VIDEO_ID}/maxresdefault.jpg`

### Google Images
Search: `"[Artist] [Song] cover art"` → Tools → Size → Large
